# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 0.7.x   | :white_check_mark: |
| < 0.7   | :x:                |

## Reporting a Vulnerability

We take security seriously. If you discover a security vulnerability, please report it privately.

### How to Report

1. **Do not** open a public issue
2. Email security concerns to: razvanazamfirei@gmail.com
3. Include:
    - Description of the vulnerability
    - Steps to reproduce
    - Potential impact
    - Suggested fix (if any)

### What to Expect

- Acknowledgment within 48 hours
- Regular updates on progress
- Credit in release notes (unless you prefer anonymity)

### Safe Harbor

We consider security research conducted in good faith to be:

- Authorized
- Legal
- Appreciated

Thank you for helping keep chezmoi-files secure!
